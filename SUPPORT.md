# Support

Use the right channel for the type of problem.

## Where To Go

- GitHub Discussions: questions, ideas, examples, and workflow tradeoffs
- GitHub Issues: bugs, docs gaps, and scoped feature requests
- Pull Requests: concrete fixes ready for review
- Security reports: follow [SECURITY.md](SECURITY.md)

## What To Include

For install or runtime problems, include:

- agent name and version if known,
- operating system,
- install command used,
- whether you installed from the repo root or a direct skill path,
- the exact error output if there is one.

For workflow questions, include:

- which skill you installed,
- which repo you are trying to use it in,
- which extension hooks you have already mapped,
- what you expected the skill to do.

## Expectations

- We aim to keep the repo welcoming and responsive.
- We do not guarantee SLA-based support for the public repo.
- Maintainers may redirect broad roadmap ideas into Discussions before turning them into issues.
