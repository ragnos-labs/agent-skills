# v0.1.2

Initial public release of the `ship` skill.

## Install

```bash
npx skills add ragnos-labs/agent-skills --skill ship
```

Pin to this release:

```bash
npx skills add https://github.com/ragnos-labs/agent-skills/tree/v0.1.2/skills/ship
```

## What's included

- Full release workflow: classify, preflight, docs sync, review, stage and commit, quality gate, publish, report
- Works with Claude Code (`/ship`) and Codex
- Extension hooks for mapping to your repo's own commands
- Rerun model for caching across phases
